Helps Developers

Este proyecto nacio con el proposito de ayudar a los desarrolladores de todo el mundo con la solucion de problemas, sugerencias de codigo, recomendaciones de libros de programacion y todo lo relacionado con este asombroso mundo del desarrollo de software.

Sistema de gestion de tareas

Este proyecto nacio con el proposito de ayudar a los desarrolladores a seguir las tareas asignadas. Por otro lado, el gerente del proyecto puede ver cada tarea y estado a traves de un sitio web, ademas, por otro lado puede descargar informes de Excel de las tareas de cada desarrollador.


Sistema de localizacion

Este proyecto nacio con el proposito de ayudar al Operador Telefonico de un centro de llamadas a registrar su informacion sobre la llamada en un sitio web y tener el control de la informacion en una base de datos. Por otro lado, esta aplicacion ayudo al Gerente del proyecto a medir la productividad del equipo a traves de informes excel y un sitio web en tiempo real donde el Gerente del proyecto podia ver la productividad del dia de cada Operadora Telefonica.


Comunidad cristiana

Este proyecto nacio con el proposito de ayudar a esta comunidad cristiana a tener un registro de cada persona para enviarles material didactico sobre la Biblia. Por otro lado, cuando esta comunidad realizaba un evento, podia tener el control de las personas que asistian para monitorear su crecimiento personal.


Aplicacion de escritorio

Este proyecto nacio con el proposito de ayudar a los directores del banco a explotar la informacion de una base con mas de diez millones de registros y entregarles esta informacion en tres formatos diferentes para que su visualizacion fuera clara.
